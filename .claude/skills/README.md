# Frontend Design Skill

この Skill は、Claude が生成するフロントエンドデザインの品質を劇的に向上させるためのものです。

## 概要

LLM にデザインを依頼すると、トレーニングデータに頻出する「ありきたりなデザイン」（Inter フォント、紫のグラデーションなど）に収束しがちです。この Skill は、そうした「AI Slop（AI の粗悪な生成物）」を避け、創造的で特徴的なフロントエンドを生成するよう Claude を導きます。

## 改善される要素

- タイポグラフィ: 独特で美しいフォント選択
- カラー＆テーマ: 統一感のある配色、強いアクセントカラー
- アニメーション: マイクロインタラクションとエフェクト
- 背景: 単色ではなく、深みと雰囲気を持つ背景

## 使い方

### Claude apps（Web 版）での使用

1. 設定画面から Skills を有効化
2. この Skill を ZIP ファイルとしてアップロード
3. フロントエンド開発タスクを依頼すると自動的に適用されます

### Claude Code（ローカル環境）での使用

1. このディレクトリを ~/.claude/skills/ にコピー
2. Claude Code を起動
3. フロントエンド開発タスクで自動的にロードされます

### API 経由での使用

1. /v1/skills エンドポイントにこの Skill をアップロード
2. Messages API リクエストで Skill を指定
3. Code Execution Tool beta を有効化

## 効果

この Skill を使用すると：

- SaaS ランディングページが洗練されたデザインに
- ブログレイアウトが独自性のあるスタイルに
- 管理ダッシュボードがプロフェッショナルな見た目に

## ライセンス

この Skill は学習・研究目的で自由に使用できます。
